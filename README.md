# HTML Practice Project

A comprehensive HTML learning project that demonstrates various HTML concepts through practical examples. This repository contains a series of lessons that progressively introduce HTML elements, attributes, and best practices.

## Project Structure

The project is organized into 10 lessons, including a final project, each building upon the previous one:

### Lesson Progression

1. **Lesson 1**: Basic HTML structure and Hello World
   - Introduction to HTML document structure
   - Basic text elements (h1, p)
   - Simple inline CSS styling

2. **Lessons 2-4**: Core HTML Elements
   - External CSS integration
   - HTML5 favicon implementation
   - Basic styling and layout

3. **Lesson 5**: Links and Navigation
   - Introduction to hyperlinks
   - Creating an About page
   - File downloads
   - Email and phone links
   - External links with target attributes

4. **Lessons 6-9**: Advanced HTML Features
   - Image handling and organization
   - Semantic HTML elements
   - Navigation menus
   - Tables and forms
   - Time elements
   - HTML5 semantic tags (header, nav, main, article, section, aside)
   - Accessibility features

5. **Lesson 10**: Final Project - Restaurant Website
   - Complete website implementation
   - Multiple pages (Home, Contact, Hours)
   - Image integration
   - Custom styling and layout

## Features Implemented

- Proper HTML5 document structure
- Meta tags for character encoding and document information
- External CSS and favicon integration
- Responsive images with proper attributes
- Semantic HTML elements for better accessibility
- Internal and external navigation
- Contact information links
- File download capabilities
- Organized file structure with separate directories for images

## Technologies Used

- HTML5
- CSS3
- SVG images for scalable graphics

## Getting Started

To explore this project:

1. Clone the repository
2. Start from Lesson 1 and progressively move through each lesson
3. Each lesson builds upon concepts from previous lessons
4. Use a modern web browser to view the HTML files
